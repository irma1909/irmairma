Portofolio

Project Terbaru

Ini adalah proyek pertama saya, sebuah aplikasi To-Do List sederhana yang memungkinkan pengguna menambahkan, mengedit, dan menghapus tugas. Aplikasi ini dibangun menggunakan HTML, CSS, dan JavaScript, dan membantu saya memahami dasar-dasar interaksi pengguna (UI) dan manipulasi DOM (Document Object Model). Melalui proyek ini, saya belajar bagaimana menyimpan data di memori lokal browser, serta memperbaiki keterampilan pemrograman dasar saya dalam pengembangan aplikasi web.

Proyek CRUD Menggunakan Filament

Proyek ini adalah implementasi sistem CRUD (Create, Read, Update, Delete) menggunakan Filament, sebuah framework admin yang berbasis Laravel. Dalam proyek ini, saya membuat antarmuka admin yang memudahkan pengguna untuk mengelola data seperti menambah, mengedit, melihat, dan menghapus entri secara efisien. Saya memanfaatkan fitur Filament untuk membangun panel admin yang interaktif dan responsif, serta mengoptimalkan pengelolaan database di backend. Melalui proyek ini, saya mendalami penggunaan Laravel dan Filament, serta memahami alur kerja dalam pengembangan aplikasi web berbasis data.

Pembuatan Tabel di Terminal

Dalam proyek ini, saya membuat tabel sederhana langsung di terminal menggunakan perintah echo dan column pada bash shell. Tabel ini menampilkan data dalam format yang rapi dengan baris dan kolom yang diatur secara manual. Proyek ini mengajarkan saya bagaimana memformat teks di terminal serta menampilkan data secara struktural, tanpa memerlukan bahasa pemrograman tambahan.

Proyek Form Login Menggunakan HTML dan CSS

Proyek ini adalah pembuatan form login sederhana menggunakan HTML dan CSS untuk tampilan antarmuka. Dalam proyek ini, saya merancang form yang responsif dan user-friendly dengan input field untuk username dan password, serta tombol untuk login. Saya menggunakan CSS untuk mengatur layout, warna, dan estetika visual dari form, termasuk efek hover dan fokus pada elemen input. Melalui proyek ini, saya memperdalam pemahaman tentang HTML form elements dan teknik CSS styling, serta mempelajari pentingnya desain yang sederhana dan intuitif dalam pembuatan UI.

Portofolio Menggunakan Tailwind CSS

Proyek ini adalah pembuatan portofolio pribadi yang dibangun menggunakan Tailwind CSS untuk menciptakan desain yang modern, responsif, dan minimalis. Saya memanfaatkan utilitas bawaan Tailwind untuk mengatur layout, tipografi, warna, dan elemen-elemen UI secara cepat dan efisien. Dengan pendekatan utility-first yang ditawarkan Tailwind, saya mampu merancang halaman portofolio yang fleksibel dan mudah disesuaikan. Proyek ini membantu saya meningkatkan pemahaman dalam penggunaan Tailwind CSS untuk membangun antarmuka web yang bersih, responsif, dan berfokus pada pengalaman pengguna (UX).
